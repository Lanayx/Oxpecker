## Empty Solid

Empty template for Oxpecker.Solid applications.

### Run this example

0. Install [.NET SDK](https://dotnet.microsoft.com/en-us/download) and [Node.js](https://nodejs.org/en)


1. Install dependencies

```bash
npm install
```

2. Run the development server

```bash
npm start
```

3. Publish the app

```bash
npm run build
```
